  1. A cosa serve bootstrap?

    Bootstrap è un framework CSS open-source che semplifica la creazione di interfacce grafiche responsive e coerenti tra i vari dispositivi. Offre componenti predefiniti e utilità per velocizzare il design di siti e applicazioni.

  2. Cosa devi importare per creare una pagina web con bootstrap

  3. Cos'è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

    Un CDN (Content Delivery Network) è una rete di server distribuiti geograficamente che fornisce file statici (come CSS e JavaScript) agli utenti, riducendo la latenza e migliorando la velocità di caricamento. I vantaggi del CDN rispetto al download diretto di Bootstrap includono:

    • Migliore velocità di caricamento: il server più vicino risponde più velocemente.
    • Cache condivisa: i file di Bootstrap vengono spesso già memorizzati nella cache del browser dell'utente.
    • Aggiornamenti automatici: si ha accesso all'ultima versione disponibile senza doverla scaricare manualmente.

  4. A cosa serve questo attributo del tag meta?

    Questo attributo fa sì che la pagina si adatti alla larghezza dello schermo del dispositivo, impostando lo zoom iniziale a 1. In altre parole, rende la pagina più responsive e ottimizzata per ogni tipo di dispositivo.

  5. Che differenza c'è tra Container e Container Fluid?

    • Container: crea un contenitore di larghezza fissa e centrato, che si adatta a vari dispositivi secondo dei punti di rottura predefiniti (breakpoints) da Bootstrap.
    • Container-fluid: crea un contenitore a larghezza piena (100% della larghezza disponibile), indipendentemente dalle dimensioni dello schermo.
